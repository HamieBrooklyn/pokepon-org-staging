Privacy Policy

Last updated: May 7, 2026

This Privacy Policy describes how Poké Pon (“we”, “us”) handles information when you use our Discord bot/application (the “Service”). By using the Service, you agree to this policy.

1. What Discord provides to the bot

When you interact with the Service on Discord, we receive information that Discord includes with events and interactions, such as:

  • Your Discord user ID, username, avatar, and server membership context needed to run commands.
  • Channel and server (guild) IDs where commands are used, when applicable.
  • Message content only if your server enables the Message Content privileged intent for the bot (used for prefix-style chat commands where enabled).

2. Data we store (game database)

To operate gameplay features, the Service may store in the operator’s database:

  • Discord user IDs and server IDs tied to game state (for example: inventory, balances, trades, duels, auctions).
  • Gameplay records and timestamps generated by your use of the Service.
  • Technical logs may be kept by the operator for security and debugging (typically short-lived).

We do not require your email address from Discord for basic gameplay. We do not use the Service to sell your personal data.

3. Monetization & purchases (if enabled)

If the operator enables Discord monetization features (for example, SKUs and entitlements), Discord may process payments and share entitlement state with the application as described in Discord’s terms and Discord’s privacy policy. We use entitlement information only to unlock described features (such as adjusted cooldowns or premium behavior configured in the bot).

4. Third-party services

The Service may request public card data from the Pokémon TCG API or similar providers. Those requests are generally not “about you” personally, but your IP address may be seen by those services as part of normal HTTPS requests from the hosting environment hosting the bot. Review their policies if you self-host or if the public instance operator changes providers.

5. Retention & deletion

Game data is retained as long as it is needed to operate the Service or as required by law. To request deletion of data associated with your Discord account for a public instance, contact the operator via the Poke-Cards repository. If you run your own copy of the bot, you control the database directly.

6. Security

We use reasonable safeguards appropriate to the nature of the Service. No online service can guarantee perfect security.

7. Children

The Service is not directed to children under the age required by Discord in your region. Use should comply with Discord’s age requirements.

8. Changes

We may update this policy from time to time. The updated version will be posted on this site with a new “Last updated” date.

9. Contact

Questions about privacy: use the contact path published in the Poke-Cards repository.
